== Description ==
--------------------------------------------------------------------
Jarvix Pay is an intelligent one-stop mobile payment solution supporting both online and face-to-face scenarios. Accept Visa, MasterCard, American Express and more directly on your store with the Jarvix Pay Payment Gateway for WooCommerce.

GROW YOUR BUSINESS EASIER THAN EVER

The Jarvix Pay plugin extends WooCommerce allowing you to take payments directly on your store via Jarvix Pay Payment API.

Jarvix Pay is available for Merchants in: 

-	Hong Kong
-	Macao

Jarvix Pay is on a mission to build an efficient and inclusive payment and financial service ecosystem, making it easy and cost-efficient for businesses of all sizes to accept payments online and access credit services, hence make it easy for everyone to grow their businesses.

WHY CHOOSE JARVIX PAY?

**1. Lightning Fast**

Merchants register an account in 3 mins, accept online payments in 5 seconds and get paid in 4 business days.

**2. No Fixed Cost**

Jarvix Pay has zero annual fee, setup fee, deposit and transaction volume requirement. You only get charged with simple and competitive service rates when you proceed payment. Payouts are transferred to your bank account on a 4-businss-day rolling basis.

**3. Report Everywhere**

Merchants export the reports (e.g. transactions, refunds, cashflow) anywhere and anytime in a mobile app.

== Installation ==
--------------------------------------------------------------------
AUTOMATIC INSTALLATION
Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t need to leave your web browser. To do an automatic install of the Jarvix Pay Payment Gateway for WooCommerce, login to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type “Jarvix Pay Payment Gateway for WooCommerce” and click Search Plugins. Once you’ve found our plugin you can view details about it such as the point release, rating, and description. Most importantly, of course, you can install it by simply clicking “Install Now”, then “Activate”.

MANUAL INSTALLATION
1. Before installation, make sure you have installed WooCommerce plugin and it is activated.
2. Upload the plugin files to the `/wp-content/plugins/jarvixpay-payment-gateway` directory, or install the plugin through the WordPress plugins screen directly.
3. Activate the plugin through the 'Plugins' screen in WordPress.
4. Use the WooCommerce ->  Settings -> Payments, and find Jarvix Pay to configure the plugin.

== Upgrade Notice ==
--------------------------------------------------------------------

Since WordPress 4.9 the minimum recommended requirement to run a WordPress site is PHP 7.2

To run WordPress we recommend your host supports:
PHP version 7.2 or greater
MySQL version 5.6 or greater OR MariaDB version 10.0 or greater
HTTPS support

Therefore start with this version the focus is only on PHP7.2+ improvements and no more backward compatibility.
Members who use older PHP versions are asked to download the last version before that one which is Version 2.7.4.3. 
We recommend asking your hosting provider to provide your account with PHP 7.2. Thanks for understanding!
